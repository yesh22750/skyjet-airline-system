# 🛫 SkyJet Airline System - Premium Edition

A comprehensive, production-ready airline management system with complete ticketing, seat booking, and integrated payment gateway.

## ✨ Key Features

- **✈️ Flight Ticketing** - Search, book, and manage flights in real-time
- **💺 Seat Selection** - Interactive 2D seat map with real-time availability
- **💳 Payment Gateway** - Secure Stripe & Razorpay integration
- **👤 User Management** - Complete authentication and profile management
- **📊 Admin Dashboard** - Flight and booking management
- **📱 Responsive UI** - Works perfectly on desktop, tablet, and mobile
- **🔐 Secure** - JWT authentication and encrypted data
- **📈 Scalable** - Built with modern technologies

## 🛠 Tech Stack

**Backend:**
- Node.js & Express
- MongoDB
- JWT Authentication
- Stripe & Razorpay APIs

**Frontend:**
- React 18
- React Router v6
- Axios
- CSS3
